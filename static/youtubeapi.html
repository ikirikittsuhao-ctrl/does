YouTube (Invidious) API プロキシの概要

提供されたJavaScriptコードは、オープンソースのYouTubeフロントエンドであるInvidiousのAPI群を踏み台（プロキシ）にして、YouTubeのデータを安全かつ高速に取得するためのEdge Runtime用APIサーバーのプログラムです。

主な機能と特徴

  • 爆速レスポンス（レース勝負）：Promise.any を使い、登録された10箇所のサーバー（インスタンス）へ同時にリクエストを送信。もっとも早く正常なデータを返した優秀なサーバーの応答を瞬時に採用します。
  • 高い安定性と冗長化：特定のインスタンスがダウンしていたり、YouTube側の規制（ボットチャレンジ等）に引っかかってHTMLを返してきたりした場合、自動的にそれを検知して除外し、生きている別のサーバーのデータを採用します。
  • リクエスト制限（セキュリティ）：正規表現 RE_TARGET により、動画詳細、コメント、検索、チャンネル情報、プレイリスト、トレンド、人気動画など、システムに必要な特定のJSONデータAPIのみを中継するよう制限されています。
  • CORS対応：Webブラウザ上のフロントエンドアプリから直接このAPIを叩けるよう、CORSヘッダー（Access-Control-Allow-Origin: *）が適切に付与されています。

技術仕様

  • 動作環境：Vercel Edge Functions や Cloudflare Workers などの Edge Runtime に最適化されています。
  • タイムアウト：5秒（5000ms）以内に反応のない遅いインスタンスは自動で通信を遮断（Abort）します。
