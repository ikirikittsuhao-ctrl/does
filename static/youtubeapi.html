YouTube (Invidious) API プロキシ仕様書

本ページは、仙人開発部門が設計したEdge Runtime（Vercel Edge Functions）上で動作する高性能YouTubeプロキシサーバーの技術リファレンスです。10箇所のグローバルインスタンスへの同時並行リクエスト（Promise.any）により、安定して超高速なJSONデータを返却します。

基本情報

  • ベースURL: https://uow-bakend.vercel.app
  • 認証/CORS: Access-Control-Allow-Origin: * (CORS制限なし。フロントエンドから直叩き可能)
  • タイムアウト制限: 各バックエンド通信は最大 5000ms で自動遮断されます。

利用可能ルート一覧（コピペ用）

正規表現 RE_TARGET により、中継を許可されているエンドポイント一覧です。右側のボタンからURL一式をコピーできます。

1. 動画情報の取得

https://uow-bakend.vercel.app/api/v1/videos/動画ID

2. 動画コメントの取得

https://uow-bakend.vercel.app/api/v1/comments/動画ID

3. 動画検索

https://uow-bakend.vercel.app/api/v1/search?q=検索ワード

4. チャンネル情報 / チャンネル動画一覧

https://uow-bakend.vercel.app/api/v1/channels/チャンネルID
https://uow-bakend.vercel.app/api/v1/channels/videos/チャンネルID

5. プレイリスト / 急上昇 / 人気動画

https://uow-bakend.vercel.app/api/v1/playlists/プレイリストID
https://uow-bakend.vercel.app/api/v1/trending
https://uow-bakend.vercel.app/api/v1/popular

レスポンス形式

正常レスポンス (200 OK)

対象インスタンスから正常にパース可能なピュアJSONが返された場合、以下のヘッダーと共にそのまま透過して出力されます。

Headers:
  Access-Control-Allow-Origin: *
  Access-Control-Allow-Methods: GET, POST, OPTIONS
  Content-Type: application/json; charset=utf-8

ターゲット外エラー (404 Not Found)

正規表現 RE_TARGET にマッチしない無効なパスへアクセスした際のレスポンスです。

{
  "error": "Not Found",
  "message": "This proxy only supports specific Invidious JSON APIs."
}

全インスタンス障害エラー (502 Bad Gateway)

登録されている10箇所すべてのInvidiousインスタンスがダウン、タイムアウト、またはYouTube側のBot Challenge規制（HTML返却）に直面した場合のレスポンスです。

{
  "error": "All Invidious instances failed or returned Bot Challenges.",
  "details": [
    "Instance https://invidious.ritoge.com returned Bot Challenge (HTML).",
    "Instance https://yt.omada.cafe returned status 500",
    "signal is aborted"
  ]
}
